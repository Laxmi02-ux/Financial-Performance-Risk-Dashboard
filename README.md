# Financial Performance Analysis (Multi-Sector)

## Problem
Comparing financial performance across companies and sectors over time can be complex. This project aims to simplify financial analysis using structured metrics.

## Dataset
- Financial data of 14 companies  
- Covers a period of 15 years (2009–2023)  

## Tools & Technologies
- Excel
- SQL
- Power BI

## Approach
- Collected and cleaned multi-year financial data  
- Analyzed key financial metrics such as profitability and leverage  
- Developed a Financial Strength Index (FSI) for comparison  
- Built dashboards for visualization  

## Key Insights
- Variation in financial performance across sectors over time  
- Differences in leverage and profitability trends among companies  
- Identification of periods with increased financial risk  

## Outcome
Delivered a dashboard and structured analysis to compare company performance and identify financial trends, making it easier to interpret large financial datasets.
